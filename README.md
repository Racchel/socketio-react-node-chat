# Chat com Socket.io

Chat em tempo real com ReactJS, NodeJS e Socket.io

---

### 🔧 Executar

Para executar esse projeto:

----

- Entre na pasta do cliente
```
cd client
```

- Instale as dependências necessárias
```
npm install
```

- Rode o script
```
npm start
```
----
Faça o mesmo processo para o servidor:

```
cd server
```

- Instale as dependências necessárias
```
npm install
```

- Rode o script
```
npm start
```

Agora, aproveite!
